Add project from vscode to github

Steps

  • Create a Folder named project_vscode_github
  • Open the folder in vscode
  • create a a file named index.html and let it opened on vscode
  • On Vscode click on Source control icon and it pops up initialize repository
  • After initialization the file index.html displayed U which means untracked file and above it with a commit message filed
  • Wrtie your fist commit message and click on a mark symbole above it.
  • create a repository in your github account named pld_javacript and copy the link, go to the vscode click on view menu, select the command pallect and search with keyword add git remote to display a url field to paste the repository url link and hit enter
  • type the name of the repository you created pld_javascript on the name add remote field
  • finaly, click on publish branch and go back to the github repository and click on it to display the file you just added from vscode to github
